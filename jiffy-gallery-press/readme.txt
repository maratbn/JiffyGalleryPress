Setup image galleries in a jiffy via shortcodes.  No locked premium features.


== Description ==

  Setup image galleries in a jiffy via shortcodes specifying image slugs, and
  have the images load in a jiffy by downloading only the minimum size image
  necessary for the client viewport size.  Images are rendered as large as
  possible within the viewport when expanded.

  No locked premium features, and tested to work on iOS devices.

Official project URLs:

  https://github.com/maratbn/JiffyGalleryPress


== Installation ==

1. Unzip contents of `jiffy-gallery-press.zip` into the directory `/wp-content/plugins/jiffy-gallery-press/`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
